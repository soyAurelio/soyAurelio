### 🧠 Aprendiendo ahora

Profundizando en sistemas distribuidos más allá del request-response:

- **NATS JetStream**: consumer groups, pull/push, acknowledgment policies, retry backoff
- **Microservicios**: bounded contexts, sagas (coreografía vs orquestación), circuit breaker, API Gateway vs service mesh
- **Observabilidad**: correlación trace-log-métrica, tail sampling por decisión, SLOs que reflejan experiencia real del usuario
